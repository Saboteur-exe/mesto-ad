# Mesto

Интерактивная страница для обмена фотографиями мест. Пользователи могут редактировать профиль, добавлять карточки с фото, ставить лайки и удалять свои карточки. Проект подключён к REST API и написан на чистом JavaScript с использованием Vite.

Выполнено в рамках курса **«Фронтенд-разработка»** (Яндекс Практикум / СГУ).

🔗 **Опубликованный проект:** https://Saboteur-exe.github.io/mesto-ad/

## Технологии

- Vanilla JavaScript (ES6+)
- Vite
- БЭМ
- REST API
- GitHub Actions (автодеплой)

## Запуск

```bash
npm install
```

### Локальный сервер

```bash
npm run dev
```

### Сборка

```bash
npm run build
```

### Публикация

```bash
npm run deploy
```

Команда делает коммит и пуш в ветку `main`. GitHub Actions автоматически собирает проект и публикует его на GitHub Pages.
