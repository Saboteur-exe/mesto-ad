# Mesto

Интерактивная страница с карточками мест. Проект написан на чистом JavaScript с использованием Vite.

🔗 **Опубликованный проект:** https://Saboteur-exe.github.io/mesto-ad/

## Запуск

```bash
npm install
```

### Локальный сервер

```bash
npm run dev
```

### Сборка

```bash
npm run build
```

### Публикация на GitHub Pages

```bash
npm run build && npx gh-pages -d dist
```
