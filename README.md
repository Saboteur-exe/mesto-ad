# Mesto

Интерактивная страница для обмена фотографиями мест. Пользователи могут редактировать профиль, добавлять карточки с фото, ставить лайки и удалять свои карточки. Проект подключён к REST API и написан на чистом JavaScript с использованием Vite.

Выполнено в рамках курса **«Фронтенд-разработка»** (Яндекс Практикум / СГУ).

🔗 **Опубликованный проект:** https://saboteur-exe.github.io/mesto-production/

## Технологии

- Vanilla JavaScript (ES6+)
- Vite
- БЭМ
- REST API
- GitHub Actions (автодеплой)

## Запуск

```bash
npm install
```

### Локальный сервер

```bash
npm run dev
```

### Сборка

```bash
npm run build
```

## Публикация

Проект публикуется автоматически через GitHub Actions при каждом push в ветку `main`.
Собранные файлы из папки `dist` деплоятся в публичный репозиторий, откуда GitHub Pages раздаёт готовый сайт.
